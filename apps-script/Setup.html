Deploy to Cloud

Now we need to create a secure URL for your iOS Shortcut to send data to.

  1. Click Deploy (blue button top right of the screen)
  2. Select New deployment
  3. Click the gear icon ⚙️ next to "Select type" and choose Web app
  4. Under Execute as, select Me
  5. Under Who has access, select Anyone
  6. Click Deploy and hit Authorize access if prompted.
  7. Copy the Web App URL it gives you.

🎉 You're almost ready!

Your Google Sheet is now a database.

Next step: Go back to the Poor Decisions website on your iPhone, click "Add to iPhone", and paste your Web App URL when the Shortcut asks for it.
